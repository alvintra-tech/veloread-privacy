\f0\fs24 \cf0 # VeloRead Privacy Policy\
\
Last updated: March 12, 2026\
\
VeloRead is a local-first PDF reading and knowledge application developed by Alvintra.\
\
This Privacy Policy explains what information VeloRead handles, how it is used, and how users can control or delete their data.\
\
VeloRead is designed to respect user privacy and minimize data collection.\
\
---\
\
# 1. Local-First Architecture\
\
VeloRead is designed as a local-first application.\
\
Most information generated while using the app remains stored only on your device and is not transmitted to external servers.\
\
This includes:\
\
\'95 highlights  \
\'95 annotations  \
\'95 generated insights  \
\'95 idea links and connections  \
\'95 reading history and statistics  \
\'95 insight review progress  \
\'95 document metadata index  \
\'95 favorites and bookmarks  \
\'95 app preferences and hint flags  \
\'95 exported notes or documents\
\
This information remains under your control on your device unless you explicitly export or share it.\
\
---\
\
# 2. Data Stored Locally on Your Device\
\
The app may store the following information locally to provide its features:\
\
\'95 PDF documents opened by the user  \
\'95 highlights and annotations created while reading  \
\'95 generated insights and document summaries  \
\'95 connections between insights  \
\'95 reading progress and reading statistics  \
\'95 insight review state and reminders  \
\'95 bookmarks and favorites  \
\'95 document metadata for navigation  \
\'95 application settings and interface preferences\
\
This data is stored locally on your device and is not transmitted to Alvintra servers.\
\
---\
\
# 3. Advertising\
\
The free version of VeloRead may display advertisements in certain parts of the application.\
\
Advertisements are provided by third-party advertising services such as:\
\
Google AdMob\
\
Advertising providers may collect limited device information such as the Advertising ID in order to deliver and measure advertisements.\
\
For more information, please review Google\'92s privacy policy:\
\
https://policies.google.com/privacy\
\
Advertisements may appear in certain free sections of the app.  \
The PDF reader interface itself does not display advertisements.\
\
---\
\
# 4. In-App Purchases\
\
VeloRead may offer optional premium features through in-app purchases.\
\
Payments and subscription management are handled by platform providers such as Google Play and services such as RevenueCat.\
\
These services may process purchase-related information according to their own privacy policies.\
\
VeloRead does not store payment card information.\
\
---\
\
# 5. Permissions Used by the App\
\
VeloRead may request certain device permissions to provide core functionality.\
\
File access  \
Allows the app to open and read PDF documents selected by the user.\
\
Internet access  \
Used for advertisements, purchase validation, update checks, and optional support links.\
\
Advertising ID  \
Used by advertising services to deliver and measure advertisements.\
\
Permissions are used only for the purposes described above.\
\
---\
\
# 6. Data Collection and Sharing\
\
VeloRead itself does not collect personal data from users or transmit reading data to external servers.\
\
However, certain third-party services integrated in the app (such as advertising providers) may collect limited technical identifiers required to provide their services.\
\
These services operate under their own privacy policies.\
\
---\
\
# 7. User Control Over Data\
\
Because VeloRead stores most information locally on your device, users maintain direct control over their data.\
\
Users can manage, export, or delete stored information directly within the application.\
\
---\
\
# 8. Data Deletion\
\
All VeloRead data is stored locally on the user's device.\
\
You can delete your data at any time by opening:\
\
Settings \uc0\u8594  Reset Data\
\
Options include:\
\
\'95 Reset Insights  \
\'95 Reset Reading History  \
\'95 Reset All App Data\
\
Resetting permanently deletes stored highlights, insights, annotations, and related reading data from your device.\
\
---\
\
# 9. No Account Required\
\
VeloRead does not require account creation.\
\
The application can be used without signing in or providing personal information.\
\

\
VeloRead uses open-source libraries.\
\
A list of open-source licenses used by the application can be viewed inside the app under:\
\
Settings \uc0\u8594  Legal\
\

\
# 11. Children's Privacy\
\
VeloRead is not directed toward children under the age of 13.\
\
The application does not knowingly collect personal information from children.\
\
